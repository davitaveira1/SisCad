# Configuração do Projeto

## 1. Credenciais do Supabase

Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:

```env
SUPABASE_URL=https://sua-url.supabase.co
SUPABASE_KEY=sua_chave_publica_aqui
```

> **Importante:** O arquivo `.env` já está no `.gitignore` e **não** será versionado. Nunca compartilhe suas credenciais publicamente.

---

## 2. Criação da Tabela

Execute o seguinte script SQL no **SQL Editor** do Supabase para criar a tabela de funcionários:

```sql
create table public.funcionarios (
  id bigint generated by default as identity primary key,
  nome text not null,
  cargo text not null,
  endereco text,
  email text unique,
  salario numeric(10,2) not null,
  created_at timestamp with time zone default now()
);
```
